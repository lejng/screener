# Commands (os windows)

### 1. Determinate is it virt env or not
```
where pip
```

### 2. Activate virtual env or create virtual env
activate
```
.venv\Scripts\activate
```
create:
```
python -m venv .venv
```

### 3. Install dependencies
```
pip install -r requirements.txt
```

### 4. How to run
```
python -m src.main_ui
```

### 5. Deactivate virtual env
```
deactivate
```
